# iron-peak-gym

Modern, minimal, multi-page gym website built with React + Tailwind.

## Tech

- React + TypeScript + Vite
- Tailwind CSS v4
- react-router-dom (multi-page routing)
- framer-motion (smooth animations)
- lucide-react (icons)

## Pages

- Home
- About
- Programs
- Schedule
- Trainers
- Pricing
- Gallery
- Contact

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
